# timeisup command

## Description

Starts timer for 50 minutes and notifies user when time is up.

## Usage

```shell
$: timeisup
```

## Installation

```shell
$: cd ~

$: git clone git@github.com:chernetsky/timeisup.git

$: ln -s /usr/bin/timeisup  ~/timeisup/start
```

## Dependencies

**notify-send** linux command
